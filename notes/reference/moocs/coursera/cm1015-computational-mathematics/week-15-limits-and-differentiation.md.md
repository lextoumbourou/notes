## Lesson 8.1. Definition of limit for a sequence

Example:

$a_n = n / (n+1)$

$0, \frac{1}{2}, \frac{2}{3}, \frac{3}{4}, ... \frac{100}{101}$

In general: $\lim_{n \rightarrow \infty} a_n = L \text{ if } \forall \epsilon > 0 \ \exists \ N: \text{ for } n > N |a_n -L| < \epsilon$
