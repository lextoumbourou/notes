The set of elements that common to 2 sets.

Represented by the $\cap$ symbol (`\cap` in [LaTeX](LaTeX.md)).

Example (using [Listing Method](set-listing-method.md)):

$A = \{a, b, c\}$
$B = \{c, d, e\}$
$A \cap B = \{c\}$

See also [Union](union.md)
