## Cross Multiply

If you have a fraction on both sides of an expression, cross-multiplication can be used to eliminate the fractions:

$\frac{a}{b} = \frac{c}{d}$
$ad = bc$

Now we can solve for any of the variables, for example:

$a = \frac{bc}{d}$
