**Agent Skills** are structured packages of Markdown files and scripts that augment [AI Agents](ai-agents.md)' capabilities. They usually look something like this:

```
~/.claude/skills/some-skill/
├── SKILL.md
└── scripts/
    └── some_script.py
```
